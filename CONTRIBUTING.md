# Contributing

As contribuições na organização ElegeAi devem preservar clareza de domínio,
rastreabilidade e coerência com a arquitetura do ecossistema.

## Diretrizes

- Prefira branches focadas e commits intencionais.
- Documente integrações, contratos e impactos entre serviços.
- Atualize o README do repositório quando a responsabilidade do domínio mudar.
- Evite acoplamentos cruzados sem justificar o custo arquitetural.

## Pull requests

- Explique o objetivo da mudança.
- Liste impactos, dependências e riscos.
- Inclua validação mínima e observações de rollout quando aplicável.
