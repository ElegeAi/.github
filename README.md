# ElegeAi `.github`

Repositório especial de **arquivos-padrão de saúde da comunidade** da organização **ElegeAi**.

Este repositório concentra os arquivos padrão que o GitHub pode aplicar automaticamente
aos demais repositórios da organização, como:

- `CODEOWNERS`
- `CONTRIBUTING.md`
- `SECURITY.md`
- `PULL_REQUEST_TEMPLATE.md`
- templates de issue

## Papel deste repositório

O objetivo aqui é padronizar colaboração, triagem e governança mínima em toda a
organização. A documentação privada e mais ampla de arquitetura e posicionamento
do produto continua centralizada em `ElegeAi/.github-private`.

## O que este repositório centraliza

- padrões de contribuição e abertura de mudanças;
- política base de reporte de segurança;
- ownership inicial da organização;
- templates de issues e pull requests;
- convenções mínimas reaproveitáveis entre os serviços.

## Escopo

Este repositório deve permanecer enxuto e focado em padrões reutilizáveis pela
organização. Documentação estratégica, técnica ou sensível deve ficar no repositório
privado de governança.

## Observação

Sempre que houver divergência entre um padrão global deste repositório e uma necessidade
específica de domínio, o repositório do serviço pode complementar a documentação local,
desde que preserve coerência com a governança organizacional.
