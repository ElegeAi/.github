# ElegeAi `.github`

Repositório especial de **community health files** da organização **ElegeAi**.

Este repositório concentra os arquivos padrão que o GitHub pode aplicar automaticamente
aos demais repositórios da organização, como:

- `CODEOWNERS`
- `CONTRIBUTING.md`
- `SECURITY.md`
- `PULL_REQUEST_TEMPLATE.md`
- templates de issue

## Papel deste repositório

O objetivo aqui é padronizar colaboração, triagem e governança mínima em toda a
organização. A documentação privada e mais ampla de arquitetura e posicionamento
do produto continua centralizada em `ElegeAi/.github-private`.

## Escopo

Este repositório deve permanecer enxuto e focado em padrões reutilizáveis pela
organização. Documentação estratégica, técnica ou sensível deve ficar no repositório
privado de governança.
